# ⚛️ Quantum Club – RVU Website

> Official website for the Quantum Club at RV University  
> Built using modern web technologies to showcase events, resources, and innovation.

---

## 🌟 About

The **Quantum Club Website** is a dynamic and interactive platform designed to:

- 🚀 Showcase club events and activities  
- 📢 Share announcements and updates  
- 📚 Provide curated learning resources  
- 🌐 Build a strong and engaging tech community  

This project reflects innovation, creativity, and technical excellence within the university ecosystem.

---

## 🛠️ Tech Stack

- ⚡ **Frontend:** React (Vite)  
- 🎨 **Styling:** CSS  
- 🎬 **Animations:** GSAP  
- 🌌 **3D Effects:** Three.js  
- 📦 **Package Manager:** npm  

---

## 📁 Project Structure

```bash
Quantum-RVU/
│
├── public/              # Static assets
├── src/
│   ├── components/      # Reusable UI components (Navbar, Timeline, etc.)
│   ├── pages/           # Pages (Home, About, Contact, Resources, Updates)
│   ├── hooks/           # Custom hooks
│   ├── assets/          # Images, logos, icons
│   └── lib/             # Utility & animation configs
│
├── index.html
├── package.json
├── package-lock.json
└── vite.config.js
```
